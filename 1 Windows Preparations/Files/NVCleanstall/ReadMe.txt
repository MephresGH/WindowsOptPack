INSTALLATION:
>-----------------<

Disable Wi-Fi/Ethernet before starting NVCleanstall
Enable Wi-Fi/Ethernet after program has started
Clcik "Refresh"
Select "Install best driver for my hardware"
Hit "Next"
Select "Recommended" (add personal options of choice if wanted; for best experience, just choose "Recommended")
Click "Next"
Select "Disable Installer Telemetry & Advertising", "Perform a Clean Installation"
Select "Show Expert Tweaks" and "Enable Message Signaled Interrupts (MSI Mode)"
Press "Next"
Install Driver as usual
Close NVIDIA Installer
Close NVCleanstall
